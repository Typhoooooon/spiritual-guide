对话系统设计（修正版）

首轮三视角展示 → 用户选择一位 → 深入对话

完整对话流程

第0步 → 用户输入问题，选择深度（温和/深度），可选指定思想家
第1步 → 系统并行生成3位思想家的回答（每条约200-300字，较短的完整三段式）
第2步 → 三条回答并排展示，用户阅读后选择一位继续对话
第3步 → 进入一对一多轮对话（剩余14轮），被放弃的两个回答折叠收起
第4步 → 15轮总上限（首轮3答算1轮），达到后对话结束、只读

首轮三答的生成逻辑

  • 用户指定了思想家时 — 该思想家的回答排在首位，另两位由系统从不同传统中选取（确保视角多样）
  • 用户未指定时 — 系统从不同传统中各选一位（如：一位宗教人物 + 一位哲学家 + 一位社会学家）
  • 三条回答并行生成 — 3个独立的 LLM 调用，彼此不可见对方内容，避免趋同
  • 首轮回答格式 — 精简版三段式，200-300字。顶部显示思想家姓名 + 头像 + 所属传统
